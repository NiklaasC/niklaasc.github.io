Bebop

Bebop: A huggable robot, designed to interact with the public.

An experiment by Rusty Squid to create a robot that is able to tell us more about the human condition.

Filmed for the 'How to Build a Robot' documentary aired on Channel 4.

  • Vacuum-formed PETG
  • Foam
  • Fabric
  • Aluminium
  • Acetyl
  • Servos
  • Raspberry Pi and Microcontrollers
  • Computer Vision
